## 5. Core loop

### Headline

`Сначала процесс. Потом AI.`

### Steps

1. `Диагностика` — разбираем, где сейчас уходит время.
2. `AI-карта` — выбираем задачи-кандидаты.
3. `Сценарий` — описываем, что делает человек, AI и автоматизация.
4. `Внедрение` — собираем рабочий прототип или систему.
5. `Обучение` — передаём инструкции вам или команде.
6. `Итерация` — улучшаем сценарии по реальной работе.

## 8. Implementation examples

Headline:

`Как это может выглядеть в работе`

Examples:

1. `SPA / салон`
   - Before: администратор отвечает на повторяющиеся вопросы.
   - System: AI-консьерж + FAQ + передача заявки.

2. `Эксперт / личный бренд`
   - Before: контент пишется рывками.
   - System: рубрики + промпты + перепаковка одного материала.

3. `Небольшая команда`
   - Before: каждый использует AI по-своему.
   - System: обучение + база промптов + правила проверки.

4. `Заявки и CRM`
   - Before: данные вручную переносятся из форм и чатов.
   - System: заявка → CRM → Telegram → черновик ответа.
